# Template email notaire

## Envoi du dossier de vente

```
Objet : Dossier de vente — [Adresse] — Réf. [Référence agence]

Maître [Nom notaire],

Suite à l'acceptation de l'offre pour le bien situé [adresse complète],
veuillez trouver ci-joint le dossier de vente complet :

Documents joints :
- Titre de propriété
- Renseignements urbanistiques (reçus le [date])
- Certificat PEB — Label [X] — n°[numéro]
- Contrôle électrique — [conforme/non conforme] — [date]
- Attestation sol — [statut]
- Extrait cadastral
- [Si Flandre] Asbestattest
- [Si Flandre] Bodemattest

Vendeur : [Nom] — [Adresse]
Acheteur : [Nom] — [Adresse]
Prix convenu : [Montant] EUR
Acompte : 10% = [Montant] EUR

Merci de nous communiquer les dates possibles pour la signature du compromis.

Cordialement,
[Agent] — [Agence]
[Téléphone]
[Email]
```

## Relance notaire (J+7 sans réponse)

```
Objet : Relance — Dossier [Adresse]

Maître [Nom],

Je me permets de revenir vers vous concernant le dossier de vente 
pour [adresse], envoyé le [date].

Pourriez-vous nous indiquer vos disponibilités pour la signature du compromis ?

Cordialement,
[Agent]
```
